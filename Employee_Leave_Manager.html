GitHub Auto-Save Settings

This app can automatically commit an up-to-date Excel file to a GitHub repository every time you add or change data, so your data lives alongside the web page on GitHub.
For security, your GitHub token is never stored. It's kept only in this browser tab's memory and is cleared when you close or refresh the page. You'll need to re-enter it each session before syncing.

How to set this up

  1. Push this HTML file to a GitHub repository, then enable it under Settings → Pages to host it.
  2. Create a fine-grained token at github.com/settings/personal-access-tokens/new, scoped only to this repository, with Contents: Read and write permission.
  3. Enter the repo owner, repo name, branch, and a file path above (the Excel file will be created automatically the first time you sync).
  4. Paste the token above and click Sync Now, or just keep working — the app auto-syncs a few seconds after every change, for as long as this browser tab stays open with the token entered.
  5. Re-enter the token each time you reopen the app, since it is intentionally not saved anywhere.
Data is saved automatically in this browser. Use Export to Excel, or GitHub Sync, to back it up.
